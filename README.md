## 📌 Project Overview

Credit card fraud has become one of the biggest challenges in digital banking. This project focuses on detecting fraudulent credit card transactions using **Machine Learning** techniques. The notebook demonstrates the complete workflow, including data preprocessing, exploratory data analysis (EDA), model building, training, and evaluation using **Logistic Regression**.

The goal is to build a model that can accurately distinguish between **legitimate** and **fraudulent** transactions.

---

## 🎯 Objectives

- Understand the credit card transaction dataset.
- Perform data cleaning and preprocessing.
- Analyze the distribution of fraudulent and genuine transactions.
- Train a Machine Learning classification model.
- Evaluate model performance using appropriate metrics.
- Predict whether a transaction is fraudulent or legitimate.

## 📊 Dataset

The dataset contains anonymized credit card transactions with two classes:

- **0 → Legitimate Transaction**
- **1 → Fraudulent Transaction**

### Features

- Transaction Features (V1–V28)
- Time
- Amount
- Class (Target Variable)
